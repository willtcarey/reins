# Assistant

The assistant is your long-lived conversation for each project. It's always available at the top of the sidebar — just click it and start talking.

Use it to manage your project: ask questions about the codebase, brainstorm ideas, plan work, or have it create tasks for you. It's the natural starting point for anything that doesn't already have a task.

Unlike task sessions, which are scoped to a specific piece of work, the assistant sticks around. You keep talking to the same conversation over time. When you want a fresh start, click "New conversation" — the old one moves into a collapsible "Previous conversations" section.
