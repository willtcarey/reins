# Progressive Web App (PWA)

Reins can be installed as a standalone app on phones and desktops using the browser's "Add to Home Screen" or install prompt.

## What it does

- **Standalone mode**: When installed, the app opens without browser chrome (address bar, tabs), feeling like a native app.
- **Theme integration**: The status bar / title bar matches the app's dark background.
- **Offline shell**: The app can launch even without a network connection. Data still requires the backend to be reachable.

## How to install

1. Open Reins in your phone's browser (or desktop Chrome/Edge).
2. Use **"Add to Home Screen"** (mobile) or the **install icon** in the address bar (desktop).
3. The app appears on your home screen / app launcher.
